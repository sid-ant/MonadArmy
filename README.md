# MonadArmy

RazorPay Hackthaon Project 

Rex is a platform to post or accept any time of jobs with guaranteed pay. 

# Run Instructions 

Install Python
Create virtual env
- python -m venv env
Run Env
- env/scripts/activate
- pip install -r requirements.txt
- export FLASK_APP=rex
- export FLASK_ENV=development
- flask init-db
- flask run

Or

Heroku
- heroku local
